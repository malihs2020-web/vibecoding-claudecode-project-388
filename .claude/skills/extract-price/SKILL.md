# Extract price

Извлекает цену товара со страницы интернет-магазина по её URL.

## Вход

Один URL страницы товара.

## Выход

Объект:

```json
{ "regular_price": 1990, "sale_price": null }
```

- `regular_price` — обычная (полная) цена товара, число.
- `sale_price` — цена по скидке, число, или `null`, если скидки нет.

Если цену на странице найти не удалось, вернуть текстовый ответ: `товар не найден`.

## Логика извлечения

1. Загрузить HTML страницы по переданному URL.
2. При необходимости распарсить HTML (например, найти элементы с ценой по типичным признакам: атрибуты/классы вида `price`, `regular-price`, `sale-price`, `old-price`, микроразметка `itemprop="price"`, теги `<meta property="product:price:amount">` и т.п.).
3. Из найденных значений убрать пробелы, неразрывные пробелы, символы валют и разделители тысяч, привести к числовому виду (число с точкой как разделителем дробной части).
4. Определить, какая цена обычная, а какая — цена со скидкой:
   - если на странице только одна цена — это `regular_price`, `sale_price` = `null`;
   - если найдены две цены (старая/зачёркнутая и новая) — старая идёт в `regular_price`, новая — в `sale_price`.
5. Если после разбора страницы ни одной цены не найдено, вернуть `товар не найден` вместо объекта.
